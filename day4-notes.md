# Day 4 - Linux Users, Groups & Processes

## What I Learned
- Linux is a multi-user operating system
- Root user has full system control
- Users and groups manage permissions and access
- Processes are running programs in the system
- top and ps commands help monitor system activity

## Commands Practiced
- whoami
- id
- cat /etc/passwd
- groups
- ps aux
- top
- kill command
- adduser and su

## Reflection
Today I learned how Linux manages users and running processes. This is essential for server management and DevOps environments.
